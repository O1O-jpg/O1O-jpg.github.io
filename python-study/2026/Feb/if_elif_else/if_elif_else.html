判断顺序

判断是互斥且有顺序的。

  • 满足<判断条件1>将不会理会<判断条件2>和<判断条件3>
  • 满足<判断条件2>，将不会理会<判断条件3>
  • <判断条件1><判断条件2><判断条件3>均不满足，进入else
  • else也可以省略不写，效果等同个独立的if判断
